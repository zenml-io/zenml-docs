Logging metrics and metadata is standardized in ZenML. The most common pattern is to use the `log_xxx` methods, e.g.:

* Log metadata to a [model](attach-metadata-to-a-model.mdx): `log_model_metadata`
* Log metadata to an [artifact](attach-metadata-to-an-artifact.mdx): `log_artifact_metadata`
* Log metadata to a [step](attach-metadata-to-steps.mdx): `log_step_metadata`
