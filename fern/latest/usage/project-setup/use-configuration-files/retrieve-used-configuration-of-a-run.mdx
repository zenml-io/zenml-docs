---
title: Find out which configuration was used for a run
subtitle: >-
  Sometimes you might want to extract the used configuration from a pipeline
  that has already run.
slug: >-
  usage/project-setup/use-configuration-files/retrieve-used-configuration-of-a-run
---


You can do this simply by loading the pipeline run and accessing its `config` attribute.

```py
from zenml.client import Client

pipeline_run = Client().get_pipeline_run("<PIPELINE_RUN_NAME>")

configuration = pipeline_run.config
```
