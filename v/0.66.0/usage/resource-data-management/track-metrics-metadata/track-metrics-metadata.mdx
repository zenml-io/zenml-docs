Logging metrics and metadata is standardized in ZenML. The most common pattern is to use the `log_xxx` methods, e.g.:

* Log metadata to a [model](/v/0.66.0/usage/resource-data-management/track-metrics-metadata/attach-metadata-to-a-model): `log_model_metadata`
* Log metadata to an [artifact](/v/0.66.0/usage/resource-data-management/track-metrics-metadata/attach-metadata-to-an-artifact): `log_artifact_metadata`
* Log metadata to a [step](/v/0.66.0/usage/resource-data-management/track-metrics-metadata/attach-metadata-to-steps): `log_step_metadata`
